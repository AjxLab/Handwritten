Handwritten
=============================

Handwritten character recognition.


## Usage
### Build
```sh
# Place target image data in tmp/
$ python build <class name>
```
### Train
```sh
$ python train.py
```


## Requirements
- macOS
- Python 3.7.4


## Installation
### Clone This Repo
```sh
$ git clone https://github.com/AjxLab/Handwritten
$ cd Handwritten
```
### Install Python Libraries
```sh
$ pip install -r lib/requirements.txt
```
